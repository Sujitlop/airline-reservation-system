# Airline Reservation & Scheduling System

A web-based airline reservation and scheduling system built with HTML, CSS, JavaScript (frontend) and Python + FastAPI + MySQL (backend).

## Features

- **Role-Based Authentication** (Passenger, Agent, Crew, Admin)
- **Passenger Dashboard** - Search/book flights, manage bookings
- **Agent Dashboard** - Assist customers, issue tickets
- **Crew Dashboard** - View schedules, report flight status
- **Admin Dashboard** - Manage flights, crew, aircraft, revenue

## Tech Stack

### Frontend
- HTML5, CSS3, Vanilla JavaScript
- Session-based authentication

### Backend (In Development)
- Python + FastAPI
- MySQL Database
- SQLAlchemy ORM
- JWT Authentication
